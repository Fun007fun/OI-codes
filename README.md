### 关于每个文件夹的内容
+ **POJ**
文件名取自POJ题库的题的URL，根据题组命名文件名，根据题号命名代码文件的名字。
+ **其他**
是我写的一些奇怪的代码，都是不属于正规题库或者我不知道属不属于正规题库的代码。
+ **clion_work_space**
是我的clion的工作空间，毕竟IDE还是方便很多，比编辑器方便，里面还有一些别的文件，是备份。
+ **codevs**
我在codevs上面写的代码。。。
+ **tools**
一些奇怪的有趣的代码
